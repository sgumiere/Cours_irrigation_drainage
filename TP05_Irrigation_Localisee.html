Exercice 1 : Caractéristique du Goutteur

Note
Durée estimée : 15 minutes

Étape 1 : Relation Débit-Pression

La caractéristique hydraulique d'un goutteur est décrite par une relation de puissance :

$$q = K \cdot H^x$$

où $q$ est le débit (L/h), $H$ est la pression (bar), $K$ est le coefficient de calibrage et $x$ est l'exposant (exposant de stabilité).

Note
Types de goutteurs :
  • Goutteur turbulent : $x \approx 0.5$ (sensible aux variations de pression)
  • Goutteur autorégulant : $x \approx 0.1$ (peu sensible aux variations de pression)

Pour ce TP, vous utiliserez deux configurations :

Configuration A (goutteur autorégulant) : $x = 0.1$, $K = 3.8$, pression nominale $1.0$ bar, plage de fonctionnement $0.5$ à $2.5$ bar.

Configuration B (goutteur turbulent) : $x = 0.5$, $K = 1.0$, plage de fonctionnement $0.5$ à $2.5$ bar.

Afficher le code R
# Caracteristique des goutteurs
H <- seq(0.5, 2.5, by=0.1)  # Pression (bar)

# Configuration A : autoregulant
K_A <- 3.8
x_A <- 0.1
q_A <- K_A * H^x_A

# Configuration B : turbulent
K_B <- 1.0
x_B <- 0.5
q_B <- K_B * H^x_B

# Graphique
png("goutteurs.png", width=800, height=600)
plot(H, q_A, type="o", col="darkgreen", lwd=2,
     xlab="Pression H (bar)", ylab="Debit q (L/h)",
     main="Caracteristique des goutteurs", ylim=c(0, 5))
lines(H, q_B, type="o", col="darkorange", lwd=2)
legend("topright", c("Autoregulant (x=0.1)", "Turbulent (x=0.5)"),
       col=c("darkgreen", "darkorange"), lwd=2)
grid()
dev.off()

cat("Debit nominal A (1.0 bar) :",
    K_A * 1.0^x_A, "L/h\n")
cat("Debit nominal B (1.0 bar) :",
    K_B * 1.0^x_B, "L/h\n")

ParamètreValeur

Espacement des goutteurs sur rampe1.5 m
Type de tuyau rampePE 16 mm (Di = 13.6 mm)
Longueur de rampe60 m (10 arbres)
Note
Le débit total de la rampe est : $Q_{rampe} = 10 \text{ arbres} \times 4 \text{ goutteurs} \times 4 \text{ L/h} = 160 \text{ L/h}$

Étape 2 : Calcul des Pertes de Charge Darcy-Weisbach

Les pertes de charge linéaires sont données par la formule de Darcy-Weisbach :

$$J = f \cdot \frac{L}{D} \cdot \frac{V^2}{2g}$$

Pour les écoulements turbulents en tuyau lisse (Blasius) :

$$f = \frac{0.316}{Re^{0.25}}$$

Pour les rampes avec sorties multiples, la perte de charge équivalente est corrigée par le facteur de Christiansen :

$$\Delta H_{rampe} = F \cdot \Delta H_{lin}, \quad F = \frac{1}{1+m} + \frac{1}{2N} + \sqrt{\frac{m-1}{6N^2}}$$

avec $m = 1.75$ (coefficient empirique) et $N$ le nombre de goutteurs.

Afficher le code R
# Parametres du systeme
Q_rampe <- 160 / 3600 / 1000  # m3/s
D_rampe <- 13.6 / 1000        # m
L_rampe <- 60                 # m
n_goutteurs <- 40
rho <- 1000                   # kg/m3
g <- 9.81                     # m/s2
mu <- 1e-3                    # Pa.s

# Vitesse et nombre de Reynolds
A_rampe <- pi * (D_rampe/2)^2
V_rampe <- Q_rampe / A_rampe
Re <- rho * V_rampe * D_rampe / mu

# Coefficient de friction (Blasius)
f <- 0.316 / Re^0.25

# Perte de charge lineaire Darcy-Weisbach
J <- f * (L_rampe / D_rampe) * (V_rampe^2 / (2*g))

# Facteur de Christiansen
m <- 1.75
F_chris <- 1/(1+m) + 1/(2*n_goutteurs) +
           sqrt((m-1)/(6*n_goutteurs^2))

# Perte de charge rampe
DH_rampe <- F_chris * J

cat("Nombre de Reynolds :", round(Re, 0), "\n")
cat("Coefficient de friction f :", round(f, 4), "\n")
cat("Perte de charge lineaire J :", round(J, 3), "m\n")
cat("Facteur de Christiansen F :", round(F_chris, 4), "\n")
cat("Perte de charge totale rampe :", round(DH_rampe, 3), "m\n")
cat("Perte de charge totale rampe :", round(DH_rampe*10.197, 3), "kPa\n")

Étape 3 : Variation de Pression le Long de la Rampe

La pression varie le long de la rampe en raison des pertes de charge. Si $H_0$ est la pression à l'entrée, la pression au point $i$ est : $H_i = H_0 - \Delta H_{0 \to i}$

Afficher le code R
# Profil de pression le long de la rampe
n_sections <- 10
H0 <- 1.5         # pression entree (bar)

# Position des goutteurs
positions <- seq(1.5, 60, by=6)

# Perte de charge par section
DH_per_section <- DH_rampe / n_sections

# Pression a chaque goutteur
H_goutteurs <- numeric(n_sections)
debit_goutteurs <- numeric(n_sections)

for(i in 1:n_sections) {
  H_bar <- H0 - i * DH_per_section
  H_goutteurs[i] <- H_bar * 10.197

  # Debit du goutteur (autoregulant) - 4 goutteurs/arbre
  debit_goutteurs[i] <- 4 * (H_bar/1.0)^0.1 * 4
}

# Graphique du profil de pression
png("profil_pression.png", width=800, height=600)
plot(positions, H_goutteurs, type="o", col="darkblue", lwd=2,
     xlab="Distance le long de la rampe (m)",
     ylab="Pression (kPa)",
     main="Profil de pression le long de la rampe")
abline(h=H0*10.197, col="green", lty=2)
grid()
dev.off()

cat("Pression entree :", H0, "bar\n")
cat("Pression a l'extremite :",
    H_goutteurs[n_sections], "kPa\n")
cat("Variation de pression relative :",
    round(100*(H0*10.197 - H_goutteurs[n_sections])/
    (H0*10.197), 1), "%\n")

Étape 4 : Vérification de l'Uniformité

L'uniformité de distribution est évaluée par deux critères :

$$CU = 100 \times \left(1 - \frac{\sum_{i=1}^{n} |q_i - q_{moy}|}{n \cdot q_{moy}}\right)$$ $$EU = 100 \times \frac{q_{min}}{q_{moy}}$$
Note
Critères d'acceptabilité (norme FAO) :
  • CU $> 90\%$ : Bonne uniformité
  • CU $75\%$–$90\%$ : Acceptable
  • CU $< 75\%$ : Non acceptable
Afficher le code R
# Calcul de l'uniformite
q_moy <- mean(debit_goutteurs)
q_min <- min(debit_goutteurs)

# Coefficient d'uniformite (CU)
CU <- 100 * (1 - sum(abs(debit_goutteurs - q_moy)) /
             (length(debit_goutteurs) * q_moy))

# Efficience d'uniformite (EU)
EU <- 100 * q_min / q_moy

cat("Debit moyen :", round(q_moy, 2), "L/h\n")
cat("Debit minimum :", round(q_min, 2), "L/h\n")
cat("Debit maximum :", round(max(debit_goutteurs), 2), "L/h\n")
cat("Coefficient d'uniformite (CU) :", round(CU, 1), "%\n")
cat("Efficience d'uniformite (EU) :", round(EU, 1), "%\n")

# Diagnostic
if(CU > 90) {
  cat("=> Bonne uniformite\n")
} else if(CU > 75) {
  cat("=> Uniformite acceptable\n")
} else {
  cat("=> Uniformite NON acceptable\n")
}

Note

Étape 1 : Dimensionnement du Porte-Rampe

Dans une plantation, plusieurs rampes sont alimentées par un porte-rampe (collecteur). Pour cette application :

ParamètreValeur
Nombre de rampes8
Débit par rampe160 L/h
Débit total porte-rampe1280 L/h
Type de tuyauPE 32 mm
Longueur porte-rampe240 m

Utilisez la méthode de Hazen-Williams pour cette conduite plus grande. Le coefficient $C$ pour PE est typiquement 150 :

$$J = \left(\frac{Q}{0.2785 \cdot C \cdot D^{2.63}}\right)^{1.852}$$

où $J$ est en m/100m, $Q$ en m$^3$/s et $D$ en m.

Afficher le code R
# Porte-rampe
Q_porte <- (1280 / 3600 / 1000)  # m3/s
D_porte <- 32 / 1000             # m
L_porte <- 240                   # m
C_HW <- 150

# Formule Hazen-Williams
J_HW <- (Q_porte / (0.2785 * C_HW * D_porte^2.63))^1.852
DH_porte <- J_HW * L_porte / 100

cat("Perte de charge porte-rampe :", round(DH_porte, 2), "m\n")
cat("Perte de charge porte-rampe :", round(DH_porte*9.81, 2), "kPa\n")

Étape 2 : Dimensionnement de la Conduite Principale

La conduite principale amène l'eau de la pompe au premier répartiteur. Étudiez plusieurs diamètres (63, 75, 90, 110 mm) et choisissez en fonction de la vitesse acceptable (0.6–1.5 m/s).

ParamètreValeur
Débit total10240 L/h
Matériau tuyauPVC
Coefficient Hazen-Williams150
Longueur500 m
Afficher le code R
# Conduite principale - comparaison de diametres
Q_principal <- 10240 / 3600 / 1000  # m3/s
L_principal <- 500
C_HW <- 150
diametres <- c(63, 75, 90, 110) / 1000

resultats <- data.frame(
  Diametre_mm = c(63, 75, 90, 110),
  Vitesse_ms = numeric(4),
  PerteMH = numeric(4),
  PertekPa = numeric(4)
)

for(i in seq_along(diametres)) {
  D <- diametres[i]
  A <- pi * (D/2)^2
  v <- Q_principal / A
  J_HW <- (Q_principal / (0.2785 * C_HW * D^2.63))^1.852
  DH <- J_HW * L_principal / 100

  resultats$Vitesse_ms[i] <- round(v, 3)
  resultats$PerteMH[i] <- round(DH, 2)
  resultats$PertekPa[i] <- round(DH * 9.81, 1)
}

print(resultats)

# Choix : vitesse entre 0.6 et 1.5 m/s
indice_choix <- which(resultats$Vitesse_ms >= 0.6 &
                      resultats$Vitesse_ms <= 1.5)
cat("Diametres acceptables : ",
    resultats$Diametre_mm[indice_choix], "mm\n")

Étape 3 : Synthèse et Bilan de la Pompe

La hauteur manométrique totale (HMT) requise par la pompe est :

$$HMT = H_g + \Delta H_{total} + H_{goutteur}$$

où $H_g$ : hauteur géométrique (différence d'élévation), $\Delta H_{total}$ : somme des pertes de charge, $H_{goutteur}$ : pression de fonctionnement du goutteur (typiquement 1.0–1.5 bar).

Afficher le code R
# Bilan de la pompe
Hg <- 15          # hauteur geometrique (m)
H_goutteur <- 1.5 # pression en bar
H_goutteur_m <- H_goutteur * 10.197

# Pertes de charge totales
