Exercice 1 : Interpolation IDW (Inverse Distance Weighting)

Durée : 20 minutes

Étape 1 : Préparation des données

Chargez les données de TP 3 ou recréez-les, puis créez une grille de prédiction.

Afficher le code R
# Charger les bibliotheques
library(gstat)
library(sp)
library(raster)

# Recreation des donnees de TP 3
set.seed(42)
n_points <- 50
x <- runif(n_points, 0, 500)
y <- runif(n_points, 0, 500)

g <- gstat(formula=z~1, locations=~x+y, dummy=TRUE, beta=5,
           model=vgm(psill=1.2, model="Exp", range=100, nugget=0.5),
           nmax=20)
ks_values <- predict(g, newdata=data.frame(x=x, y=y), nsim=1)$sim1
ks_values <- abs(ks_values) + 2

data_ks <- data.frame(x=x, y=y, Ks=ks_values)
coordinates(data_ks) <- ~x + y

# Creer une grille de prediction 100x100
grid <- expand.grid(x = seq(0, 500, length.out=100),
                     y = seq(0, 500, length.out=100))
coordinates(grid) <- ~x + y
gridded(grid) <- TRUE

cat("Nombre de cellules de prediction :", nrow(grid), "\n")
Résultat attendu
Vous devez obtenir une grille régulière de 10 000 cellules (100 × 100) couvrant le domaine [0, 500] × [0, 500] m.

Exercice 2 : Krigeage ordinaire

Durée : 30 minutes

Étape 1 : Rappel du variogramme ajusté

Recréez le modèle de variogramme obtenu en TP 3.

Afficher le code R
# Calculer le variogramme experimental
vg_exp <- variogram(Ks ~ 1, data = data_ks)

# Definir et ajuster le modele spherique
sill_est <- max(vg_exp$gamma)
range_est <- vg_exp$dist[which(vg_exp$gamma >= 0.95 * sill_est)[1]]
nugget_est <- vg_exp$gamma[1]

vgm_sph_init <- vgm(psill = sill_est, model = "Sph",
                     range = range_est, nugget = nugget_est)
vgm_fit <- fit.variogram(vg_exp, vgm_sph_init)

cat("Modele de variogramme :\n")
print(vgm_fit)
Note
Le modèle de variogramme spécifie la structure spatiale de la variabilité de Ks. Il est essentiel pour le krigeage car il contrôle les poids donnés à chaque observation.

# Krigeage ordinaire
krige_result <- krige(Ks ~ 1, locations = data_ks,

# Visualisation des predictions

# Carte des predictions

      col = heat.colors(100),
      xlab = "X (m)", ylab = "Y (m)")
points(data_ks, pch = "+", cex = 2, col = "blue")
contour(krige_result["var1.pred"], add = TRUE, levels = 8,
        col = "gray20", drawlabels = TRUE)

# Carte de la variance de krigeage
image(krige_result["var1.var"],
      main = "Variance de krigeage (incertitude)",
      col = terrain.colors(100),
      xlab = "X (m)", ylab = "Y (m)")

par(mfrow = c(1, 1), mar = c(5, 4, 4, 2) + 0.1)
Résultat attendu
Vous devez obtenir deux cartes : une montrant les valeurs prédites de Ks (couleurs chaudes indiquant les valeurs élevées) et une montrant la variance de krigeage, avec l'incertitude la plus faible près des points de mesure.

Exercice 3 : Application à la gestion de l'irrigation

Durée : 20 minutes

Étape 1 : Classification en zones de gestion

Découpez la grille en trois zones de gestion selon le niveau de Ks.

Afficher le code R
# Extraire les predictions de krigeage
ks_pred <- krige_result$var1.pred

# Seuils de classification
seuil_bas <- quantile(ks_pred, 0.33, na.rm=TRUE)
seuil_haut <- quantile(ks_pred, 0.67, na.rm=TRUE)

cat("Seuils de classification :\n")
cat("  Ks faible : < ", round(seuil_bas, 2), " cm/h\n")
cat("  Ks moyen : ", round(seuil_bas, 2), " a ",
    round(seuil_haut, 2), " cm/h\n")
cat("  Ks eleve : > ", round(seuil_haut, 2), " cm/h\n")

# Creer une grille de classification
zones <- ks_pred * 0
zones[ks_pred <= seuil_bas] <- 1
zones[ks_pred > seuil_bas & ks_pred <= seuil_haut] <- 2
zones[ks_pred > seuil_haut] <- 3

# Visualisation
par(mar = c(3, 3, 3, 1))
colors_zones <- c("red", "yellow", "green")
image(matrix(zones, nrow = 100),
      col = colors_zones,
      main = "Zones de gestion de l'irrigation",
      xlab = "X (m)", ylab = "Y (m)")
points(data_ks, pch = "+", cex = 2, col = "blue")
legend("topright",
       c("Faible Ks", "Moyen Ks", "Eleve Ks"),
       col = colors_zones, pch = 15)
par(mar = c(5, 4, 4, 2) + 0.1)

# Statistiques par zone
for (z in 1:3) {
  zone_name <- c("Faible", "Moyen", "Eleve")[z]
  zone_vals <- ks_pred[zones == z]
  zone_area <- length(zone_vals) * (500/100)^2
  cat(sprintf("Zone %s : %.1f ha (%.1f%%)\n",
              zone_name, zone_area / 10000,
              100 * length(zone_vals) / length(ks_pred)))
}
Résultat attendu
Vous devez obtenir une carte répartissant le domaine en trois couleurs (rouge, jaune, vert) représentant les trois zones de Ks. Les statistiques montrent la superficie de chaque zone.

Note
Ces recommandations sont basées sur la relation entre texture du sol (inférée de Ks) et la capacité de rétention d'eau. Les zones à faible Ks retiennent mieux l'eau (souvent argile) tandis que les zones à élevé Ks drainent rapidement (sables).
Attention
Ces recommandations sont des directives générales. Les ajustements finaux doivent tenir compte d'autres facteurs : profondeur de l'eau disponible, salinité, culture, climat local.

Étape 3 : Optimisation de l'échantillonnage

Identifiez où ajouter de nouveaux points de mesure pour réduire l'incertitude.

Afficher le code R
# Extraire la variance de krigeage
var_krige <- krige_result$var1.var
std_krige <- sqrt(var_krige)

# Identifier les zones de forte incertitude
high_uncertainty_threshold <- quantile(std_krige, 0.85, na.rm=TRUE)
high_unc_cells <- which(std_krige > high_uncertainty_threshold)

# Convertir indices de cellule en coordonnees
grid_coords <- coordinates(grid)
unc_coords <- grid_coords[high_unc_cells, ]

cat("Nombre de cellules avec haute incertitude :",
    length(high_unc_cells), "\n")

# Proposer de nouveaux points d'echantillonnage
library(stats)
if (nrow(unc_coords) > 10) {
  kmeans_result <- kmeans(unc_coords, centers = 5)
  new_points <- kmeans_result$centers
} else {
  new_points <- unc_coords
}

cat("Nombre de nouveaux points proposes :", nrow(new_points), "\n")

# Visualisation
par(mar = c(3, 3, 3, 1))
image(matrix(std_krige, nrow = 100),
      main = "Zones de haute incertitude",
      col = terrain.colors(100),
      xlab = "X (m)", ylab = "Y (m)")
points(data_ks, pch = 20, cex = 2, col = "blue")
points(new_points[, 1], new_points[, 2],
       pch = 3, cex = 3, col = "red", lwd = 2)
legend("topright",
       c("Points existants", "Points proposes"),
       pch = c(20, 3), col = c("blue", "red"))
par(mar = c(5, 4, 4, 2) + 0.1)
