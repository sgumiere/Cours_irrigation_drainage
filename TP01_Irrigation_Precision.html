Étape 2 : Calcul des paramètres intermédiaires

Avant de calculer ET$_0$, nous devons calculer plusieurs paramètres intermédiaires : pression atmosphérique, constante psychrométrique, pression de vapeur saturante et pente de la courbe de pression de vapeur. L'altitude d'Agadir est d'environ 20 m.

Afficher le code R
# Parametres de base
z <- 20  # altitude en metres
latitude <- 30.4  # latitude Agadir en degres Nord
agadir$T <- (agadir$Tmin + agadir$Tmax) / 2

# Pression atmospherique (kPa)
agadir$P <- 101.3 * ((293 - 0.0065 * z) / 293) ^ 5.26

# Constante psychrometrique (kPa/C)
agadir$gamma <- 0.000665 * agadir$P

# Pression de vapeur saturante (kPa)
agadir$es <- 0.6108 * exp(17.27 * agadir$T / (agadir$T + 237.3))

# Pente Delta de la courbe de saturation (kPa/C)
agadir$Delta <- 4098 * agadir$es / (agadir$T + 237.3) ^ 2

# Pression de vapeur actuelle (kPa)
agadir$ea <- (agadir$HR / 100) * agadir$es

# Deficit de saturation
agadir$es.ea <- agadir$es - agadir$ea

# Vitesse du vent a 2m (m/s)
agadir$u2 <- agadir$Vent

cat("Parametres intermediaires calcules\n")
print(head(agadir[, c("T", "P", "gamma", "es", "Delta", "ea")]))

Étape 3 : Calcul du rayonnement net

Le rayonnement net $R_n$ est la différence entre le rayonnement solaire net (courtes longueurs d'onde) et le rayonnement longues longueurs d'onde. Nous utilisons les équations FAO-56.

Afficher le code R
# Constantes
Gsc <- 0.0820  # constante solaire (MJ/m2/min)
pi <- 3.14159265

# Angle de declination et distance relative Terre-Soleil
jour_moyen <- c(15, 46, 74, 105, 135, 166, 196, 227, 258, 288, 319, 350)
agadir$b <- 2 * pi * (jour_moyen - 1) / 365
agadir$delta <- 0.409 * sin(agadir$b - 1.39)
agadir$dr <- 1.00011 + 0.034221 * cos(agadir$b) +
              0.00128 * sin(agadir$b) + 0.000719 * cos(2*agadir$b) +
              0.000077 * sin(2*agadir$b)

# Angle horaire au coucher du soleil
lat_rad <- latitude * pi / 180
agadir$ws <- acos(-tan(lat_rad) * tan(agadir$delta))

# Rayonnement extraterrestre Ra (MJ/m2/jour)
agadir$Ra <- 24 * 60 / pi * Gsc * agadir$dr *
             (agadir$ws * sin(lat_rad) * sin(agadir$delta) +
              cos(lat_rad) * cos(agadir$delta) * sin(agadir$ws))

# Rayonnement solaire Rs (MJ/m2/jour)
a_s <- 0.25
b_s <- 0.5
agadir$N <- 24 / pi * agadir$ws
agadir$Rs <- (a_s + b_s * (agadir$Insol / agadir$N)) * agadir$Ra

# Rayonnement net courtes ondes Rns (MJ/m2/jour)
alpha <- 0.23
agadir$Rns <- (1 - alpha) * agadir$Rs

# Rayonnement net longues ondes Rnl (MJ/m2/jour)
sigma <- 4.903e-9
agadir$Rnl <- sigma * ((agadir$Tmax + 273.16) ^ 4 +
              (agadir$Tmin + 273.16) ^ 4) / 2 *
              (0.34 - 0.14 * sqrt(agadir$ea)) *
              (1.35 * agadir$Rs / agadir$Ra - 0.35)

# Rayonnement net
agadir$Rn <- agadir$Rns - agadir$Rnl

cat("Rayonnement net calcule\n")
print(head(agadir[, c("Ra", "Rs", "Rns", "Rnl", "Rn")]))

Étape 4 : Calcul de ET$_0$ Penman-Monteith FAO-56

L'équation de Penman-Monteith standardisée par la FAO pour calculer l'évapotranspiration de référence (ET$_0$) est :

$$\text{ET}_0 = \frac{0.408\,\Delta\,(R_n - G) + \gamma\,\frac{900}{T+273}\,u_2\,(e_s - e_a)}{\Delta + \gamma(1 + 0.34\,u_2)}$$

où les unités sont : ET$_0$ en mm/jour, $R_n$ en MJ/m$^2$/jour, $G$ = flux de chaleur du sol (MJ/m$^2$/jour), $\gamma$ = constante psychrométrique (kPa/°C), $\Delta$ = pente de saturation (kPa/°C), $u_2$ = vitesse du vent à 2 m (m/s), $T$ = température moyenne (°C).

Afficher le code R
# Flux de chaleur du sol G (MJ/m2/jour)
agadir$G <- 0  # approximation simplifiee

# Calcul de ET0 Penman-Monteith
agadir$ET0 <- (0.408 * agadir$Delta * (agadir$Rn - agadir$G) +
               agadir$gamma * (900 / (agadir$T + 273)) * agadir$u2 *
               agadir$es.ea) /
              (agadir$Delta + agadir$gamma * (1 + 0.34 * agadir$u2))

# Arrondir a 2 decimales
agadir$ET0 <- round(agadir$ET0, 2)

cat("ET0 Penman-Monteith calculee\n")
print(agadir[, c("mois", "T", "Rn", "u2", "es.ea", "ET0")])

P <- c(18, 12, 8, 5, 2, 1, 0, 1, 4, 12, 20, 22)

# Durée en jours pour chaque mois
jours_mois <- c(31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31)

# Efficience du systeme d'irrigation (localisee)

agadir$Bb_olivier <- round(agadir$Bn_olivier / Ea, 1)
agadir$Bb_agrumes <- round(agadir$Bn_agrumes / Ea, 1)

Exercice 4 : Programme d'irrigation de l'olivier (20 min)

Pour programmer l'irrigation, nous déterminons la dose par apport (dose), la fréquence d'irrigation (interval entre irrigations) en fonction de la capacité de rétention en eau du sol.

Étape 1 : Paramètres du sol

Pour un sol limono-sableux typé de la région Souss-Massa cultivé en olivier.

Afficher le code R
# Parametres du sol
CC <- 0.32      # Capacite au champ (fraction vol.)
PF <- 0.18      # Point de fletrissement permanent
Zr <- 1.2       # Profondeur racinaire (m)
f <- 0.5        # Seuil de depletition d'eau facilement disponible (50%)
Da <- 1.35      # Densite apparente (kg/m3)

# Reserve utile RFU (mm)
RFU <- (CC - PF) * Zr * 1000  # en mm
cat("Reserve facile RFU:", RFU, "mm\n")

# Dose nette (eau facilement disponible)
dose_nette <- RFU * f  # en mm
cat("Dose nette (50% RFU):", dose_nette, "mm\n")

Étape 2 : Fréquence et dose d'irrigation

La fréquence d'irrigation (nombre de jours entre deux irrigations) est déterminée par le rapport : fréquence = dose / ETc.

Afficher le code R
# Calcul de la frequence d'irrigation (jours)
ETc_moyen_ete <- mean(agadir$ETc_olivier[c(6, 7, 8)])  # juin a aout
frequence <- dose_nette / ETc_moyen_ete

cat("ETc moyen ete (olivier):", round(ETc_moyen_ete, 3), "mm/jour\n")
cat("Frequence d'irrigation:", round(frequence, 1), "jours\n")

# Arrondir a 1, 2 ou 3 jours selon la pratique
frequence_arrondie <- 3  # tous les 3 jours en ete

# Dose d'irrigation
dose_irrigation <- ETc_moyen_ete * frequence_arrondie

cat("Dose d'irrigation proposee:", round(dose_irrigation, 1), "mm\n")

Étape 3 : Calendrier d'irrigation mensuel

Calculez le nombre d'irrigations, la dose totale et le volume total par mois.

Afficher le code R
# Calendrier d'irrigation (olivier, approche simple)
agadir$freq_jours <- c(
  30, 30, 25, 20, 7, 5, 3, 3, 5, 10, 20, 30
)  # frequence (jours) par mois

agadir$nb_irrig <- floor(jours_mois / agadir$freq_jours)
agadir$dose_irrig <- agadir$ETc_olivier * agadir$freq_jours
agadir$volume_total <- agadir$ETc_olivier * jours_mois

# Pour 1 hectare
agadir$vol_ha <- agadir$volume_total * 10  # 1 mm = 10 m3/ha

cat("Calendrier d'irrigation OLIVIER (1 hectare)\n")
calendrier <- data.frame(
  Mois = agadir$mois,
  Freq = agadir$freq_jours,
  Nb.Irrig = agadir$nb_irrig,
  Dose = round(agadir$dose_irrig, 1),
  Vol.m3 = round(agadir$vol_ha, 0)
)
print(calendrier)

cat("\nVolume annuel pour 1 ha:", sum(calendrier$Vol.m3), "m3\n")
Attention
Ce calendrier est une approximation. En pratique, il faut :
  1. Monitorer l'humidité du sol (tensiomètre, capteur capacitif)
  2. Ajuster selon la météo (pluies imprévues, vents forts)
  3. Tenir compte de la croissance saisonnière et du stade de floraison

  1. Calculer l'évapotranspiration de référence (ET$_0$) avec l'équation de Penman-Monteith

  • Olivier : 800–900 mm/an (culture peu exigeante)
  • Agrumes : 1000–1200 mm/an (très demandeurs d'été)

Une irrigation localisée (goutte-à-goutte) permet d'atteindre 85–95% d'efficience, contre 50–60% pour l'irrigation par sillon.
