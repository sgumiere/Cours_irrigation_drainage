Exercice 1 : Courbe de rétention van Genuchten (30 min)

La courbe de rétention en eau du sol décrit la relation entre la teneur en eau volumique $\theta$ et le potentiel matriciel $h$. Le modèle de van Genuchten est l'un des modèles les plus utilisés.

Trois sols types sont considérés : sable du Souss, limon du Saïss et argile du Gharb.

# Parametres van Genuchten - Sable Souss
theta_r_sable <- 0.05
theta_s_sable <- 0.40
alpha_sable <- 0.025
n_sable <- 2.68

# Calcul de m
m_sable <- 1 - 1 / n_sable

# Fonction van Genuchten pour theta(h)
vg_theta <- function(h, theta_r, theta_s, alpha, n) {
  m <- 1 - 1 / n
  Se <- 1 / (1 + (alpha * abs(h)) ^ n) ^ m
  theta <- theta_r + (theta_s - theta_r) * Se
  return(theta)
}

# Matrice potentiels matriciels (h de 0 a -15000 cm = pF 6)
h_sable <- seq(0, -15000, by = -100)

# Calcul theta pour tous les h
theta_sable <- vg_theta(h_sable, theta_r_sable, theta_s_sable,
                        alpha_sable, n_sable)

# Affichage
cat("Sable Souss - Courbe de retention\n")
resultat_sable <- data.frame(
  h_cm = h_sable[seq(1, length(h_sable), by = 50)],
  theta = theta_sable[seq(1, length(h_sable), by = 50)]
)
print(head(resultat_sable, 10))

Exercice 2 : Conductivité hydraulique K(h) avec Mualem (20 min)

La conductivité hydraulique diminue rapidement lorsque le sol s'assèche. Le modèle de Mualem-van Genuchten relie K(h) aux propriétés de rétention.

Étape 1 : Modèle de Mualem

La saturation effective (Se) est définie par :

$$S_e = \frac{\theta - \theta_r}{\theta_s - \theta_r}$$

Et la conductivité hydraulique relative par :

$$K_r(S_e) = S_e^{l} \left[ 1 - \left(1 - S_e^{1/m}\right)^m \right]^2$$

où $l = 0.5$ est le paramètre de tortuosité et $K(h) = K_r(S_e) \times K_s$ (Ks = conductivité à saturation).

Étape 2 : Code R pour Mualem

Afficher le code R
# Fonction Mualem-van Genuchten pour K(h)
mualem_K <- function(h, theta_r, theta_s, alpha, n, Ks) {
  m <- 1 - 1 / n

  # Saturation effective
  Se <- 1 / (1 + (alpha * abs(h)) ^ n) ^ m
  Se <- pmax(Se, 0.001)

  # Conductivite relative
  l <- 0.5  # parametre de tortuosite
  Kr <- Se ^ l * (1 - (1 - Se ^ (1 / m)) ^ m) ^ 2

  # Conductivite hydraulique K(h)
  K <- Kr * Ks

  return(K)
}

# Parametres Ks (cm/h) pour les 3 sols
sols_prop <- data.frame(
  nom = c("Sable (Souss)", "Limon (Saiss)", "Argile (Gharb)"),
  theta_r = c(0.05, 0.06, 0.10),
  theta_s = c(0.40, 0.45, 0.50),
  alpha = c(0.025, 0.008, 0.003),
  n = c(2.68, 1.89, 1.25),
  Ks = c(50.0, 2.5, 0.3)
)

# Calcul K(h) pour les 3 sols
h_values <- seq(0, -15000, by = -100)

K_sable <- mualem_K(h_values, 0.05, 0.40, 0.025, 2.68, 50.0)
K_limon <- mualem_K(h_values, 0.06, 0.45, 0.008, 1.89, 2.5)

Exercice 3 : Simulation de l'infiltration Green-Ampt (30 min)

Le modèle Green-Ampt décrit l'infiltration en fonction du temps et permet de déterminer le moment où le ruissellement commence (ponding time).

Étape 1 : Paramètres et équations Green-Ampt

Le modèle Green-Ampt suppose un front d'humidification net séparant le sol saturé du sol initialement sec. La vitesse d'infiltration instantanée est :

$$f(t) = K_s \left[ 1 + \frac{\psi_f \Delta\theta}{F(t)} \right]$$

où $F(t)$ est l'infiltration cumulée (mm), $\psi_f$ le potentiel matriciel au front d'humidification (cm), et $\Delta\theta$ le changement de teneur en eau au front.

Paramètres Green-Ampt pour limon du Saïss
ParamètreValeur
$K_s$ (conductivité saturée)2.5 cm/h = 25 mm/h
$\psi_f$ (potentiel au front)-10 cm
$\theta_i$ (humidité initiale)0.15 m$^3$/m$^3$
$\theta_s$ (humidité saturation)0.45 m$^3$/m$^3$
$\Delta\theta$0.30 m$^3$/m$^3$

Étape 2 : Algorithme de résolution itérative

On résout numériquement F(t) par une boucle temporelle.

Afficher le code R
# Resolution Green-Ampt par iteration

# Fonction vitesse d'infiltration f(t)
f_infiltration <- function(F, Ks, psi_f, Delta_theta) {
  if (F < 0.001) F <- 0.001
  f <- Ks + (Ks * psi_f * Delta_theta) / F
  return(f)
}

# Simulation temporelle
dt <- 1  # pas de temps en minutes
t_max <- 120
time <- seq(0, t_max, by = dt)

# Initialisation des resultats
n_steps <- length(time)
F_cumul <- numeric(n_steps)
f_inst <- numeric(n_steps)
ponding_time <- NA

# Boucle temporelle
F_cumul[1] <- 0
for (i in 2:n_steps) {
  dF_mm <- (I_mmh / 60) * dt

  f_prev <- f_infiltration(pmax(F_cumul[i-1], 0.1),
                           Ks_mmh, psi_f, Delta_theta)

  if (I_mmh > f_prev && is.na(ponding_time)) {
    ponding_time <- time[i-1]
  }

  if (I_mmh > f_infiltration(F_cumul[i-1], Ks_mmh, psi_f,
                             Delta_theta)) {
    F_cumul[i] <- F_cumul[i-1]
  } else {
    F_cumul[i] <- F_cumul[i-1] + dF_mm
    if (F_cumul[i] < 0) F_cumul[i] <- 0
  }

  f_inst[i] <- f_infiltration(pmax(F_cumul[i], 0.1),
                              Ks_mmh, psi_f, Delta_theta)
}

cat("Simulation Green-Ampt realisee\n")
cat("Ponding time:", round(ponding_time, 1), "min\n")
cat("Infiltration a t=120 min:", round(F_cumul[n_steps], 1), "mm\n")

# Analyse de sensibilite : intensites differentes
intensites <- c(10, 20, 30, 40, 50)
resultats_sensibilite <- data.frame(
  Intensite_mmh = intensites,
  Ponding_time_min = numeric(length(intensites)),
  Infiltr_120min = numeric(length(intensites))
)

for (idx in 1:length(intensites)) {
  I_test <- intensites[idx]
  F_test <- numeric(n_steps)
  F_test[1] <- 0
  ponding_test <- NA

  for (i in 2:n_steps) {
    dF <- (I_test / 60) * dt
    f_calc <- f_infiltration(pmax(F_test[i-1], 0.1),
                             Ks_mmh, psi_f, Delta_theta)

    if (I_test > f_calc && is.na(ponding_test)) {
      ponding_test <- time[i-1]
    }

    if (I_test > f_calc) {
      F_test[i] <- F_test[i-1]
    } else {
      F_test[i] <- F_test[i-1] + dF
    }
  }

  resultats_sensibilite$Ponding_time_min[idx] <- if (is.na(ponding_test))
    Inf else ponding_test
  resultats_sensibilite$Infiltr_120min[idx] <- F_test[n_steps]
}

cat("\nAnalyse de sensibilite\n")
print(resultats_sensibilite)
