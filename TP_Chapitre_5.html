TP 5 — Dimensionnement hydraulique des drains

Objectifs pédagogiques
  • Appliquer l'équation de Manning pour le dimensionnement des drains latéraux en écoulement uniforme partiellement pleins.
  • Vérifier par Darcy-Weisbach (+ Colebrook) et comparer les diamètres requis.
  • Dimensionner un collecteur par itération avec le facteur de sécurité CPVQ (0,75 pour les collecteurs, 1,0 pour latéraux).
  • Vérifier les vitesses (autocurage \(v_{min}=0{,}15\) m/s ; érosion \(v_{max}=1{,}5\) m/s).
  • Sélectionner une enveloppe/filtre à partir de la granulométrie (règle de Terzaghi).
Prérequis R 
library(tidyverse)
library(scales)
Équations de base

Manning (plein ou partiellement plein, conduit circulaire) :

\[ Q = \frac{1}{n}\,A\,R_h^{2/3}\,S_0^{1/2} \]

Pour une section circulaire pleine : \(A = \pi D^2/4\) et \(R_h = D/4\), donc

\[ Q_{plein} = \frac{\pi}{4 n}\left(\frac{D}{4}\right)^{2/3} D^2 \sqrt{S_0} \;=\; \frac{\pi}{4 n}\cdot \frac{D^{8/3}}{4^{2/3}}\sqrt{S_0}. \]

Darcy-Weisbach :

\[ \Delta h = f\,\frac{L}{D}\,\frac{v^2}{2g}, \qquad \frac{1}{\sqrt f} = -2\log_{10}\!\left(\frac{\varepsilon/D}{3{,}7}+\frac{2{,}51}{Re\sqrt f}\right) \]

Valeurs typiques (drains PVC annelés / lisses) : \(n = 0{,}015\) à 0,018 ; \(\varepsilon \approx 0{,}25\) mm pour PVC annelé, 0,0015 mm pour PVC lisse.

Exercice 1 — Débit de projet d'un latéral 10 min

Un champ de 4,2 ha est drainé par des latéraux parallèles (écartement \(L=18\) m, longueur \(L_{lat}=180\) m). Le taux de drainage de projet vaut \(q = 12\) mm/j.

  1. Calculez le débit spécifique \(q\) en m/s puis en m\(^3\)/s par mètre de drain.
  2. Déduisez le débit cumulé \(Q_{lat}\) en fin de latéral (en L/s).
  3. Exprimez ce débit en \(\ell\)/s/ha et comparez à la plage typique Québec (0,5–1,5 L/s/ha).
Solution 
q_mm_j <- 12
q_m_s  <- q_mm_j / 1000 / 86400     # 1.389e-7 m/s

L      <- 18      # écartement (m)
L_lat  <- 180     # longueur latéral (m)

# Débit spécifique par mètre de drain : q * L  (apport linéique)
q_lin  <- q_m_s * L                 # m^3/s par m de drain
Q_lat  <- q_lin * L_lat             # m^3/s total en fin de latéral
Q_lat_Ls <- Q_lat * 1000            # L/s

# Surface drainée par latéral
A_ha   <- L * L_lat / 10000         # ha
Q_Ls_ha <- Q_lat_Ls / A_ha

cat(sprintf("q = %.3e m/s\n", q_m_s))
cat(sprintf("Q_lat = %.3f L/s  (%.2f L/s/ha)\n", Q_lat_Ls, Q_Ls_ha))

Résultat : \(Q_{lat}\approx 0{,}45\) L/s, soit \(\approx 1{,}39\) L/s/ha — cohérent avec la plage CPVQ (0,5–1,5 L/s/ha).

Exercice 2 — Diamètre par Manning 12 min

Le latéral de l'exercice précédent a une pente \(S_0 = 0{,}002\) m/m et un coefficient \(n = 0{,}017\). Les diamètres commerciaux disponibles sont 75, 100, 125, 150, 200 mm.

  1. Écrivez une fonction Q_full(D, n, S0) qui calcule le débit à pleine section.
  2. Pour chaque diamètre, calculez \(Q_{plein}\) et le ratio \(Q_{lat}/Q_{plein}\).
  3. Sélectionnez le plus petit diamètre tel que ce ratio reste \(\le 1\) (latéral, FS=1,0).
Solution 
Q_full <- function(D, n = 0.017, S0 = 0.002){
  A  <- pi * D^2 / 4
  Rh <- D / 4
  (1/n) * A * Rh^(2/3) * sqrt(S0)
}

D_comm <- c(0.075, 0.100, 0.125, 0.150, 0.200)
tab <- tibble(D = D_comm,
              Qp_Ls = Q_full(D_comm) * 1000,
              ratio = 0.45 / (Q_full(D_comm) * 1000))

print(tab, digits = 3)
D_choisi <- tab$D[which(tab$ratio <= 1)[1]]
cat("Diamètre choisi :", 1000*D_choisi, "mm\n")

Diamètre retenu : 75 mm (ratio 0,37, soit une section seulement à 37% d'utilisation, ce qui laisse une marge amplement suffisante).

Exercice 3 — Comparaison Manning vs Darcy-Weisbach 13 min

Pour \(D = 75\) mm, \(L=180\) m, \(Q=0{,}45\) L/s, comparez la perte de charge prédite par Manning et par Darcy-Weisbach + Colebrook (\(\varepsilon = 0{,}25\) mm, eau à 10 °C : \(\nu = 1{,}31\times10^{-6}\) m\(^2\)/s).

Solution 
D  <- 0.075
L  <- 180
Q  <- 0.45e-3
A  <- pi*D^2/4
v  <- Q/A
Rh <- D/4
nu <- 1.31e-6
Re <- v*D/nu
eps <- 0.25e-3

# Manning : inverser pour S
n  <- 0.017
S_man <- (Q*n/(A*Rh^(2/3)))^2
dh_man <- S_man * L

# Colebrook (itération)
colebrook <- function(Re, eps, D, tol = 1e-8, itmax = 60){
  f <- 0.02
  for(k in 1:itmax){
    f_new <- (1/(-2*log10(eps/(3.7*D) + 2.51/(Re*sqrt(f)))))^2
    if(abs(f_new - f) < tol) break
    f <- f_new
  }
  f
}
f  <- colebrook(Re, eps, D)
dh_dw <- f * (L/D) * v^2/(2*9.81)

cat(sprintf("v = %.3f m/s ; Re = %.0f\n", v, Re))
cat(sprintf("Manning  : S = %.4f m/m, dh = %.3f m\n", S_man, dh_man))
cat(sprintf("D-W      : f = %.4f, dh = %.3f m\n", f, dh_dw))

Résultat : \(v\approx 0{,}10\) m/s, \(Re\approx 5\,900\) (régime turbulent de transition). Manning et DW prédisent des pertes très proches (\(\approx 0{,}36\) m vs 0,38 m). Écart \(<\) 10 %, confirmant la validité de Manning en drainage.

Remarque : la vitesse 0,10 m/s est inférieure à \(v_{min} = 0{,}15\) m/s — risque de sédimentation à surveiller.

Exercice 4 — Dimensionnement d'un collecteur (itératif) 15 min

Un collecteur reçoit 28 latéraux (débit unitaire 0,45 L/s, soit \(Q_{coll} = 12{,}6\) L/s). Pente \(S_0 = 0{,}0015\), \(n=0{,}017\). Pour un collecteur, le facteur de sécurité CPVQ exige \(Q_{design}/Q_{plein} \le 0{,}75\).

  1. Calculez la plage de diamètres commerciaux (100, 125, 150, 200, 250, 300 mm) qui respectent la contrainte.
  2. Vérifiez la vitesse à pleine section (\(v_{min}=0{,}15\) m/s ? \(v_{max}=1{,}5\) m/s ?).
  3. Produisez un graphique \(D\) vs vitesse et \(D\) vs ratio de remplissage, avec la zone d'acceptabilité ombrée.
Solution 
Q_coll <- 12.6e-3   # m^3/s
S0 <- 0.0015
n  <- 0.017

D_comm <- c(0.100, 0.125, 0.150, 0.200, 0.250, 0.300)
tab <- tibble(D = D_comm) |>
  mutate(A   = pi*D^2/4,
         Rh  = D/4,
         Qp  = (1/n)*A*Rh^(2/3)*sqrt(S0),
         ratio = Q_coll / Qp,
         v_plein = Qp / A)

print(tab, digits = 3)
D_coll <- tab$D[which(tab$ratio <= 0.75)[1]]
cat("Diamètre collecteur :", 1000*D_coll, "mm\n")

# graphique
ggplot(tab, aes(D*1000, ratio)) +
  geom_hline(yintercept = 0.75, colour = "red", linetype = 2) +
  geom_line() + geom_point() +
  labs(x = "Diamètre (mm)", y = "Q_design / Q_plein",
       title = "Dimensionnement collecteur — critère CPVQ 0,75") +
  theme_minimal()

Exercice 5 — Sélection d'une enveloppe par règle de Terzaghi 10 min

Granulométrie d'un sol (passants cumulés) :

sol <- tibble(
  d_mm = c(0.002, 0.005, 0.010, 0.020, 0.050, 0.100, 0.200, 0.500, 1.0, 2.0),
  pct  = c(    8,    14,    22,    35,    52,    68,    82,    94, 98, 100)
)

Règle : une enveloppe est nécessaire si \(D_{60} < 0{,}6\) mm et si la fraction fine (\(d<0{,}075\) mm) dépasse 50 %. Critères enveloppe : \(d_{85,filtre}/d_{15,sol} > 5\) et \(d_{15,filtre}/d_{15,sol} < 5\).

  1. Tracez la courbe granulométrique en échelle log.
  2. Par interpolation, déterminez \(d_{15}\), \(d_{50}\), \(d_{60}\) du sol.
  3. Évaluez la nécessité d'une enveloppe. Si oui, proposez une plage de \(d_{15,filtre}\).
Solution 
sol <- tibble(
  d_mm = c(0.002, 0.005, 0.010, 0.020, 0.050, 0.100, 0.200, 0.500, 1.0, 2.0),
  pct  = c(    8,    14,    22,    35,    52,    68,    82,    94, 98, 100)
)

# interpolation linéaire en log(d)
interp <- approxfun(sol$pct, log10(sol$d_mm))
d15 <- 10^interp(15)
d50 <- 10^interp(50)
d60 <- 10^interp(60)
cat(sprintf("d15 = %.4f mm\nd50 = %.4f mm\nd60 = %.4f mm\n", d15, d50, d60))

# fines < 0,075 mm (interpolation pct à 0,075)
interp_pct <- approxfun(log10(sol$d_mm), sol$pct)
pct_fines <- interp_pct(log10(0.075))
cat(sprintf("%% fines (<0.075 mm) = %.1f\n", pct_fines))

# plage d15_filtre
d15f_min <- 1 * d15     # sécurité
d15f_max <- 5 * d15
cat(sprintf("d15_filtre ∈ [%.3f ; %.3f] mm\n", d15f_min, d15f_max))

ggplot(sol, aes(d_mm, pct)) +
  geom_line() + geom_point() +
  scale_x_log10() +
  labs(x = "Diamètre (mm, log)", y = "% passant",
       title = "Granulométrie du sol drainé") + theme_minimal()

Résultats : \(d_{15}\approx 0{,}006\) mm, \(d_{50}\approx 0{,}045\) mm, \(d_{60}\approx 0{,}067\) mm. Fraction fine \(\approx 61\%\) et \(d_{60}<0{,}6\) mm : enveloppe requise. La plage de \(d_{15,filtre}\) se situe entre 0,006 et 0,03 mm — en pratique on retiendra une enveloppe synthétique (PP100/PP450) plutôt qu'un gravier (difficile à trouver si fin).
