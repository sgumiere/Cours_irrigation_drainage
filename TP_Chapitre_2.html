Objectifs pédagogiques
  • Appliquer la loi de Darcy à différentes géométries (colonne, gradient 2D).
  • Calculer la conductivité équivalente pour un sol stratifié (moyennes arithmétique et harmonique).
  • Corriger la conductivité pour la température via la viscosité cinématique.
  • Quantifier l'anisotropie \(K_h/K_v\) et ses conséquences sur le design.
Prérequis R 
install.packages(c("tidyverse","ggplot2"))
library(tidyverse)
Rappel théorique

La loi de Darcy en 1D pour un milieu saturé :

\[ q = -K\,\frac{\mathrm{d}h}{\mathrm{d}z} \quad\Longleftrightarrow\quad Q = K \cdot A \cdot i \]

Pour un sol stratifié :

\[ K_h^{eq} = \frac{\sum_i e_i K_i}{\sum_i e_i} \;\;(\text{flux // couches}),\qquad K_v^{eq} = \frac{\sum_i e_i}{\sum_i e_i/K_i} \;\;(\text{flux} \perp \text{couches}) \]

Correction de température (Hazen) : \( K(T) = K_{20}\cdot \dfrac{\mu_{20}}{\mu_T} \) où \(\mu\) est la viscosité dynamique.

Contexte

Vous caractérisez un site agricole du Centre-du-Québec : un profil de sol de 3 horizons distincts est documenté, et vous disposez de mesures de conductivité saturée faites au perméamètre à charge constante en laboratoire, puis au trou à la tarière au champ.

Exercice 1 — Loi de Darcy en colonne 10 min

Une colonne de sable de 40 cm de long et 5 cm de diamètre est soumise à une charge hydraulique différentielle \(\Delta h = 12\) cm. On mesure \(V = 220\) mL en 30 s.

  1. Calculez le débit \(Q\) en m³/s puis en m/j.
  2. Déduisez la conductivité hydraulique \(K\) (m/j).
  3. Vérifiez que l'écoulement est laminaire (Re < 1) : \( Re = q\,d_{50}/\nu \) avec \(d_{50} = 0{,}5\) mm et \(\nu = 10^{-6}\) m²/s.
Solution 
L  <- 0.40          # m
D  <- 0.05          # m
A  <- pi * D^2 / 4  # m²
dh <- 0.12          # m
V  <- 220e-6        # m³
t  <- 30            # s

Q <- V / t                   # m³/s
Q_mj <- Q * 86400            # m³/j
q <- Q / A                   # vitesse darcienne m/s

i <- dh / L                  # gradient (-)
K <- q / i                   # m/s
K_mj <- K * 86400
cat(sprintf("Q   = %.3e m³/s  =  %.3f m³/j\n", Q, Q_mj))
cat(sprintf("q   = %.3e m/s\n", q))
cat(sprintf("K   = %.3e m/s  =  %.3f m/j\n", K, K_mj))

# Reynolds
d50 <- 0.5e-3
nu  <- 1e-6
Re  <- q * d50 / nu
cat(sprintf("Re  = %.3f (< 1 : Darcy applicable)\n", Re))

On attend typiquement \(K \approx 1\)–2 m/j pour un sable fin, Re ≪ 1 confirmant la validité de Darcy.

Exercice 2 — Profil stratifié et moyennes équivalentes 15 min

Voici le profil pédologique :

HorizonÉpaisseur (m)K (m/j)Texture
Ap0,301,80loam
Bt0,500,12argile
C1,202,50sable loameux
  1. Calculez \(K_h^{eq}\) et \(K_v^{eq}\).
  2. Calculez le ratio d'anisotropie \(K_h/K_v\).
  3. Quelle moyenne retenir pour : (a) dimensionner l'espacement de drains latéraux ? (b) analyser la percolation verticale ?
Solution 
sol <- tibble(
  horizon = c("Ap","Bt","C"),
  e       = c(0.30, 0.50, 1.20),   # m
  K       = c(1.80, 0.12, 2.50)    # m/j
)

Kh_eq <- with(sol, sum(e * K) / sum(e))
Kv_eq <- with(sol, sum(e) / sum(e / K))

cat(sprintf("Kh_eq = %.3f m/j  (moyenne arithmétique pondérée)\n", Kh_eq))
cat(sprintf("Kv_eq = %.3f m/j  (moyenne harmonique pondérée)\n", Kv_eq))
cat(sprintf("Kh/Kv = %.1f\n", Kh_eq/Kv_eq))

Interprétation :

  • \(K_h \approx 1{,}7\) m/j, dominé par les horizons Ap et C.
  • \(K_v \approx 0{,}3\) m/j, limité par la couche argileuse Bt.
  • Anisotropie ≈ 6 — valeur typique pour un sol loameux stratifié.
  • Espacement (flux latéral) → on utilise \(K_h^{eq}\).
  • Percolation verticale (recharge, lessivage) → on utilise \(K_v^{eq}\).

Exercice 3 — Correction de température 10 min

Au laboratoire à 22 °C, on mesure \(K = 0{,}95\) m/j. Au champ, la température du sol est de 6 °C au printemps.

  1. Implémentez la viscosité dynamique en fonction de la température (formule empirique) : \[ \mu(T) = 2{,}414\cdot 10^{-5} \cdot 10^{247{,}8/(T+133{,}15)} \quad (\text{Pa·s}) \]
  2. Corrigez \(K\) pour la température du champ.
  3. Tracez \(K(T)\) entre 0 et 25 °C et discutez l'impact pour le drainage printanier au Québec.
Solution 
mu <- function(T){
  2.414e-5 * 10^(247.8/(T + 133.15))
}

K_lab <- 0.95                      # m/j @ 22 °C
T_lab <- 22
T_champ <- 6
K_champ <- K_lab * mu(T_lab) / mu(T_champ)
cat(sprintf("K(6°C)  = %.3f m/j   (vs %.2f en labo)\n", K_champ, K_lab))

T_seq <- seq(0, 25, by = 1)
K_seq <- K_lab * mu(T_lab) / mu(T_seq)

library(ggplot2)
ggplot(data.frame(T = T_seq, K = K_seq), aes(T, K)) +
  geom_line(color = "#003e71", linewidth = 1.2) +
  geom_hline(yintercept = K_lab, linetype = "dashed", color = "#961428") +
  geom_vline(xintercept = 6, linetype = "dotted", color = "#006642") +
  labs(title = "K en fonction de la température",
       x = "Température (°C)", y = "K (m/j)") +
  theme_minimal()

Observation : à 6 °C, \(K\) est typiquement 40 % plus faible qu'à 22 °C. Conséquence : au printemps, la capacité réelle du drainage est inférieure à la capacité nominale. Il est prudent d'appliquer un facteur de sécurité de ~1,4 pour les calculs de dimensionnement.

Exercice 4 — Débit dans un réseau d'écoulement 13 min

On utilise un réseau d'écoulement (flownet) pour estimer le débit sous un barrage agricole (largeur 30 m). Le réseau a été tracé avec 5 canaux d'écoulement et 12 chutes équipotentielles. Charge totale \(\Delta H = 4\) m, sol homogène \(K = 0{,}35\) m/j.

\[ Q = K \cdot \Delta H \cdot \frac{N_f}{N_d} \cdot L \]
  1. Calculez \(Q\) (m³/j).
  2. Estimez la vitesse darcienne moyenne si la section traversée moyenne est de 4 m².
  3. Quelle précaution prendre si \(K\) n'est connu qu'à ±50 % ?
Solution 
K  <- 0.35       # m/j
dH <- 4          # m
Nf <- 5
Nd <- 12
L  <- 30         # m

Q <- K * dH * (Nf/Nd) * L
cat(sprintf("Q = %.1f m³/j\n", Q))

A <- 4
q <- Q / A
cat(sprintf("q = %.2f m/j = %.2e m/s\n", q, q/86400))

# Incertitude ±50 %
Qmin <- 0.5 * Q
Qmax <- 1.5 * Q
cat(sprintf("Fourchette Q : %.1f — %.1f m³/j\n", Qmin, Qmax))

Commentaire : le débit est proportionnel à \(K\) ; une incertitude ±50 % sur \(K\) se propage directement sur \(Q\). Dans un contexte de barrage, on utilise une estimation haute (\(K_{max}\)) pour dimensionner les enrochements anti-érosion.

Exercice 5 — Cartographie Monte-Carlo de \(K\) 12 min

Vous avez 50 mesures de \(K\) sur un plan de 40 ha suivant une loi log-normale (\(\mu_{\ln K} = -0{,}8\), \(\sigma_{\ln K} = 0{,}9\)). Simulez-les et comparez la moyenne arithmétique, géométrique et harmonique.

Solution 
set.seed(2026)
K_sample <- exp(rnorm(50, mean = -0.8, sd = 0.9))

moyennes <- tibble(
  type = c("Arithmétique","Géométrique","Harmonique"),
  K    = c(mean(K_sample),
           exp(mean(log(K_sample))),
           1/mean(1/K_sample))
)
print(moyennes)

# Histogramme log
ggplot(data.frame(K = K_sample), aes(K)) +
  geom_histogram(bins = 15, fill = "#00549b", color = "white") +
  scale_x_log10() +
  geom_vline(data = moyennes, aes(xintercept = K, color = type), linewidth = 1.1) +
  labs(title = "Distribution log-normale de K",
       x = "K (m/j, échelle log)", y = "Fréquence") +
  theme_minimal()

Leçon : \(K_{arith} > K_{géom} > K_{harm}\). Pour un agencement hétérogène typique, c'est la moyenne géométrique qui est la meilleure estimation du K effectif bloc (justification par la théorie d'homogénéisation). La moyenne arithmétique surestime systématiquement.
