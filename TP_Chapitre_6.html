Définitions

Stress Day Index (Hardjoamidjojo & Skaggs) :

\[ SDI = \sum_{j=1}^{N} SD_j\cdot CS_j \]

où \(SD_j\) est le nombre de jours avec nappe trop haute (ex. \(< 30\) cm) pondéré par un coefficient de stade \(CS_j\).

Fonction de production simple :

\[ Y = Y_{max}\,(1-\alpha\cdot SDI) \]

Coût annuel équivalent (CRF avec \(i\) et \(n\) ans) :

\[ C_{an}(L) = \frac{C_{inv}(L)\cdot i(1+i)^n}{(1+i)^n-1} + C_{ent} \]

Bénéfice annuel :

\[ B_{an}(L) = \bigl[Y(L) - Y_0\bigr]\cdot P_{prod}\cdot S \]

Exercice 1 — Courbe h/q (critère d'intensité critique) 10 min

Pour un sol loameux drainé par Hooghoudt (\(K = 0{,}8\) m/j, \(d = 3{,}2\) m, \(D = 1{,}2\) m), on souhaite tracer l'écartement requis en fonction du taux de drainage \(q\) pour trois hauteurs critiques de nappe \(h \in \{0{,}3 ; 0{,}5 ; 0{,}8\}\) m.

  1. Utilisez l'équation explicite \(L^2 = (4 K h^2 + 8 K d h)/q\).
  2. Tracez \(L\) (en ordonnée, log) vs \(q\) (en abscisse, de 5 à 25 mm/j) pour les trois h.
  3. Marquez le point de design CPVQ : \(q = 12\) mm/j, \(h = 0{,}30\) m.
Solution 
K <- 0.8; d <- 3.2; D <- 1.2
L_hoog <- function(q_mm_j, h){
  q <- q_mm_j / 1000                    # m/j
  sqrt((4*K*h^2 + 8*K*d*h) / q)
}

grid <- expand_grid(q = seq(5, 25, 1),
                    h = c(0.30, 0.50, 0.80)) |>
  mutate(L = L_hoog(q, h))

ggplot(grid, aes(q, L, colour = factor(h))) +
  geom_line(size = 1) +
  scale_y_log10() +
  geom_point(data = tibble(q=12, h=0.30, L=L_hoog(12,0.30)),
             aes(q, L), colour = "red", size = 3, inherit.aes = FALSE) +
  annotate("text", x=13, y=L_hoog(12,0.30), label="CPVQ (12 mm/j; h=30 cm)",
           hjust=0, colour="red") +
  labs(x="q (mm/j)", y="L (m, log)", colour="h (m)",
       title="Écartement requis – critère h/q") + theme_minimal()

Résultats indicatifs :

h (m)L à q=12 mm/j (m)
0,30\(\sqrt{(4\cdot 0{,}8\cdot 0{,}09 + 8\cdot 0{,}8\cdot 3{,}2\cdot 0{,}3)/0{,}012}\approx 23{,}0\)
0,50\(\approx 30{,}4\)
0,80\(\approx 38{,}9\)

Plus on tolère une nappe haute (h grand), plus on peut élargir les drains ; le design CPVQ (12 mm/j, h=30 cm) fixe \(L\approx 23\) m.

Exercice 2 — Calcul du SDI à partir de chroniques de nappe 12 min

On dispose de mesures horaires de profondeur de nappe (m sous la surface) pour la saison mai–août (122 jours). Les coefficients de stade \(CS\) varient (semis, végétation, maturation).

set.seed(2026)
N  <- 122
dt <- seq.Date(as.Date("2026-05-01"), by = "day", length.out = N)
wt <- 0.7 + 0.5*sin(2*pi*(1:N)/20) + rnorm(N, 0, 0.15)  # profondeur nappe (m)
CS <- c(rep(0.6, 25), rep(1.0, 60), rep(0.4, 37))        # semis, végét., maturité
chronique <- tibble(date = dt, wt_m = wt, CS = CS)
  1. Calculez \(SD_j = \max(0,\,0{,}30 - wt_j)\) (jours de stress, en m·j) : nappe au-dessus de 30 cm.
  2. Calculez \(SDI = \sum SD_j \cdot CS_j\).
  3. Identifiez la période la plus critique (fenêtre mobile 7 jours).
Solution 
library(zoo)
set.seed(2026)
N  <- 122
dt <- seq.Date(as.Date("2026-05-01"), by = "day", length.out = N)
wt <- 0.7 + 0.5*sin(2*pi*(1:N)/20) + rnorm(N, 0, 0.15)
CS <- c(rep(0.6, 25), rep(1.0, 60), rep(0.4, 37))

chronique <- tibble(date = dt, wt_m = wt, CS = CS) |>
  mutate(SD   = pmax(0, 0.30 - wt_m),
         SDcs = SD * CS,
         SDcs_7d = rollmean(SDcs, 7, fill = NA, align = "right"))

SDI <- sum(chronique$SDcs)
cat(sprintf("SDI saisonnier = %.2f m·j\n", SDI))

pire <- chronique |> slice_max(SDcs_7d, n = 1)
cat("Pire fenêtre 7 j :", format(pire$date), "\n")

ggplot(chronique, aes(date, wt_m)) +
  geom_line() +
  geom_hline(yintercept = 0.30, linetype = 2, colour = "red") +
  scale_y_reverse() +
  labs(x = NULL, y = "Profondeur nappe (m)",
       title = "Chronique de nappe (positif = vers le bas)") + theme_minimal()

Avec un SDI saisonnier exprimé en m·j, on peut hiérarchiser les parcelles : SDI < 1 m·j signifie drainage efficace. La fenêtre de stress maximal correspond en général à la fin juin (stades végétatifs + précipitations).

Exercice 3 — Optimum économique (écartement) 15 min

Une analyse coût-bénéfice porte sur un champ de 20 ha (maïs, \(Y_{max}\) = 12 t/ha, prix \(P\) = 220 \$/t). Le drainage dure 40 ans, taux d'intérêt \(i=5\%\), entretien 30 \$/ha·an. Relation \(Y(L)\) estimée par simulation DRAINMOD :

perf <- tibble(
  L_m       = c(10, 15, 20, 25, 30, 40, 50),
  Y_tha     = c(11.8, 11.6, 11.3, 10.9, 10.4, 9.4, 8.2),
  cout_in   = c(4800, 3600, 2900, 2450, 2150, 1750, 1500)  # $/ha
)
  1. Calculez le coût annuel équivalent par hectare (CRF + entretien).
  2. Calculez le bénéfice net annuel : \(B = (Y - Y_0)\cdot P - C_{an}\), avec \(Y_0 = 7{,}0\) t/ha (sans drainage).
  3. Déterminez \(L^\star\) optimal. Tracez \(B(L)\).
Solution 
i <- 0.05; n <- 40
CRF <- i*(1+i)^n / ((1+i)^n - 1)
Cent <- 30
Y0 <- 7.0; P <- 220

perf <- tibble(
  L_m     = c(10, 15, 20, 25, 30, 40, 50),
  Y_tha   = c(11.8, 11.6, 11.3, 10.9, 10.4, 9.4, 8.2),
  cout_in = c(4800, 3600, 2900, 2450, 2150, 1750, 1500)
) |>
  mutate(C_an = cout_in * CRF + Cent,
         gain = (Y_tha - Y0) * P,
         B    = gain - C_an)

print(perf, digits = 3)
L_star <- perf$L_m[which.max(perf$B)]
cat("Écartement optimal L* =", L_star, "m\n")

ggplot(perf, aes(L_m, B)) + geom_line() + geom_point() +
  geom_vline(xintercept = L_star, linetype=2, colour="red") +
  labs(x="L (m)", y="Bénéfice net ($/ha·an)",
       title="Optimum économique") + theme_minimal()

Résultat indicatif :

L (m)\(C_{an}\) ($/ha·an)Gain ($/ha·an)B
10309,81056746,2
15239,81012772,2
20198,9946747,1
25172,7858685,3

L\(^\star\) = 15 m (B = 772 \$/ha·an). Noter la faible pente de B autour de l'optimum : une marge ±5 m est acceptable (robustesse au prix du grain).

Exercice 4 — Drainage contrôlé : volume drainé 13 min

Comparez deux modes d'opération sur la chronique ci-dessous : (A) drainage libre — évacuation tant que nappe \(>\) drain ; (B) contrôlé — évacuation tant que nappe \(>\) 60 cm (hauteur de retenue).

set.seed(42)
jours <- 1:120
wt    <- 0.2 + 0.6*sin(2*pi*jours/30) + rnorm(120, 0, 0.1)  # profondeur (m)
q_cap <- 0.012                                               # m/j (capacité drain)
  1. Pour chaque jour, \(q_A = q_{cap}\) si wt < 1,2 m (drain à 1,2 m), 0 sinon.
  2. \(q_B = q_{cap}\) si wt < 0,60 m, 0 sinon.
  3. Comparez les volumes cumulés (mm·ha cumulés = mm·10).
  4. Bonus : estimez la réduction de pertes de nitrates (hypothèse : 15 mg/L).
Solution 
set.seed(42)
jours <- 1:120
wt    <- 0.2 + 0.6*sin(2*pi*jours/30) + rnorm(120, 0, 0.1)
q_cap <- 0.012
prof_drain <- 1.2
retenue    <- 0.60

scen <- tibble(j = jours, wt = wt) |>
  mutate(qA = ifelse(wt < prof_drain, q_cap, 0),
         qB = ifelse(wt < retenue,    q_cap, 0),
         VA = cumsum(qA)*1000,     # mm cumulés
         VB = cumsum(qB)*1000)

cat(sprintf("Volume drainé total A = %.1f mm\n", tail(scen$VA,1)))
cat(sprintf("Volume drainé total B = %.1f mm\n", tail(scen$VB,1)))
cat(sprintf("Réduction = %.0f %%\n", 100*(1 - tail(scen$VB,1)/tail(scen$VA,1))))

# nitrates : 15 mg/L = 0.015 kg/m^3
# 1 mm sur 1 ha = 10 m^3 d'eau
dV_mm <- tail(scen$VA,1) - tail(scen$VB,1)
dN    <- dV_mm * 10 * 0.015        # kg/ha
cat(sprintf("Réduction pertes NO3 ≈ %.1f kg/ha/saison\n", dN))

scen |> pivot_longer(c(VA,VB)) |>
  ggplot(aes(j, value, colour=name)) + geom_line(size=1) +
  labs(x="Jour", y="Volume drainé cumulé (mm)",
       colour="Scénario", title="Drainage libre vs contrôlé") +
  theme_minimal()

Interprétation : le contrôlé réduit le volume drainé de 30–40 %, ce qui se traduit par une réduction proportionnelle des pertes en nitrates (quelques kg·ha/saison) — un co-bénéfice environnemental important pour les programmes québécois Prime-Vert et Agri-Climat.

Exercice 5 — Sensibilité multi-critères (heatmap) 10 min

Pour le maïs, générez une heatmap du bénéfice \(B(L, P)\) où \(L \in [10,40]\) m et \(P \in [150, 300]\) \$/t.

Solution 
# fonction Y(L) interpolée
Y_fun <- approxfun(c(10,15,20,25,30,40,50),
                   c(11.8,11.6,11.3,10.9,10.4,9.4,8.2))
C_fun <- approxfun(c(10,15,20,25,30,40,50),
                   c(4800,3600,2900,2450,2150,1750,1500))

grid <- expand_grid(L = seq(10, 40, 2),
                    P = seq(150, 300, 10)) |>
  mutate(Y = Y_fun(L), C = C_fun(L),
         B = (Y - 7)*P - (C*0.05*(1.05)^40/((1.05)^40-1) + 30))

ggplot(grid, aes(L, P, fill = B)) +
  geom_raster() +
  geom_contour(aes(z = B), colour = "white") +
  scale_fill_viridis_c() +
  labs(x="L (m)", y="Prix maïs ($/t)", fill="B ($/ha·an)",
       title="Sensibilité : Bénéfice = f(L, prix)") + theme_minimal()

Observation : l'optimum se déplace légèrement vers les petits L quand le prix monte (intensification rentable) et vers les grands L quand le prix baisse (minimiser l'investissement). Le design reste stable entre 15 et 20 m pour la plage typique du prix au Québec.
