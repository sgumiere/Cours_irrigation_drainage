Prérequis R 
library(tidyverse)
Équations clés

Hooghoudt (régime permanent) :

\[ L^2 = \dfrac{8 K_h d_e H + 4 K_v H^2}{q} \]

avec \(d_e\) la profondeur équivalente (dépend de \(L\) et \(D_0\)) :

\[ d_e = \dfrac{D_0}{1 + \dfrac{8 D_0}{\pi L}\,\ln\!\left(\dfrac{D_0}{r_e}\right)} \]

Glover-Dumm (transitoire) :

\[ h(t) = h_0 \exp\!\left(-\dfrac{\pi^2 K \bar{D} t}{\mu L^2}\right) \]

où \(\bar{D}\) est l'épaisseur moyenne saturée, \(\mu\) la porosité de drainage.

Exercice 1 — Hooghoudt : résolution itérative 15 min

Conditions : \(K = 0{,}6\) m/j (isotrope), \(D_0 = 3\) m, \(H = 0{,}5\) m, \(q = 10\) mm/j, \(r_e = 0{,}10\) m (drain enveloppé).

  1. Implémentez une fonction hooghoudt(L) qui retourne \(L_{calc}\) à partir de \(L\) (itération par point fixe).
  2. Trouvez \(L\) solution (tolérance 0,01 m).
  3. Vérifiez que \(d_e < D_0\).
Solution 
K  <- 0.6
D0 <- 3
H  <- 0.5
q  <- 0.010
re <- 0.10

d_e <- function(L, D0, re){
  D0 / (1 + (8*D0)/(pi*L) * log(D0/re))
}

hooghoudt_L <- function(L, K, D0, H, q, re){
  de <- d_e(L, D0, re)
  sqrt( (8*K*de*H + 4*K*H^2) / q )
}

L <- 20                # valeur initiale
for (i in 1:30){
  L_new <- hooghoudt_L(L, K, D0, H, q, re)
  if (abs(L_new - L) < 0.01) break
  L <- L_new
}
cat(sprintf("Convergence en %d itérations\n", i))
cat(sprintf("L  = %.2f m\n", L_new))
cat(sprintf("de = %.3f m (vs D0 = %.1f m)\n", d_e(L_new, D0, re), D0))

On attend \(L \approx 21\) m, \(d_e \approx 1{,}6\) m. Remarque : \(d_e\) diverge rapidement de \(D_0\) à mesure que \(L/D_0\) diminue — c'est la correction de résistance radiale de Hooghoudt.

Exercice 2 — Ernst pour sol bicouche 12 min

Sol : argile (0–1 m, \(K_1 = 0{,}15\) m/j) sur sable (1–3 m, \(K_2 = 2{,}0\) m/j). Drain à 1,2 m. \(q = 12\) mm/j, \(H = 0{,}6\) m. On utilise une version simplifiée :

\[ L^2 = \dfrac{8 K_2 D_2 H + 4 K_1 H^2}{q} \]

avec \(D_2\) l'épaisseur de la couche perméable inférieure.

  1. Calculez \(L\) avec Ernst simplifié.
  2. Comparez à ce qu'on aurait obtenu en négligeant la couche sable (\(K\) uniforme = 0,15 m/j).
Solution 
ernst_simple <- function(K1, K2, D2, H, q){
  sqrt( (8*K2*D2*H + 4*K1*H^2) / q )
}

K1 <- 0.15
K2 <- 2.0
D2 <- 1.8          # épaisseur sable sous le drain (de 1.2 à 3 m)
H  <- 0.6
q  <- 0.012

L_ernst <- ernst_simple(K1, K2, D2, H, q)
cat(sprintf("Ernst : L = %.2f m\n", L_ernst))

# Si on ignore le sable
L_uni <- sqrt( (8*K1*D2*H + 4*K1*H^2) / q )
cat(sprintf("Sans couche sable : L = %.2f m\n", L_uni))
cat(sprintf("Ratio = %.2f\n", L_ernst/L_uni))

Enseignement : la couche de sable sous-jacente multiplie le pouvoir drainant d'un facteur ~3–4. Reconnaître cette structure en phase de reconnaissance est financièrement déterminant (moins de drains nécessaires).

Exercice 3 — Glover-Dumm : rabattement transitoire 15 min

CPVQ exige qu'en 48 h, la nappe passe de 0,9 m à 0,3 m sous la surface. Données : \(K = 0{,}4\) m/j, \(\bar{D} = 2\) m, \(\mu = 0{,}07\).

  1. Trouvez \(L\) qui satisfait ce critère.
  2. Tracez \(h(t)\) pour trois espacements (\(L_{opt}\), \(L_{opt}+5\), \(L_{opt}-5\)).
Solution 
K  <- 0.4
Db <- 2
mu <- 0.07
h0 <- 0.90
hT <- 0.30
Tc <- 2              # jours

# Inversion analytique : h(T)/h0 = exp(-pi^2 K D t / (mu L^2))
# => L^2 = pi^2 K D T / (mu * ln(h0/hT))
L_opt <- sqrt( pi^2 * K * Db * Tc / (mu * log(h0/hT)) )
cat(sprintf("L optimal (Glover-Dumm) = %.2f m\n", L_opt))

glover <- function(t, h0, K, Db, mu, L){
  h0 * exp(-pi^2 * K * Db * t / (mu * L^2))
}

t <- seq(0, 5, by = 0.1)
df <- expand.grid(t = t, L = c(L_opt-5, L_opt, L_opt+5)) |>
  mutate(h = glover(t, h0, K, Db, mu, L),
         L = factor(round(L,1)))

library(ggplot2)
ggplot(df, aes(t, h, color = L)) +
  geom_line(linewidth = 1.1) +
  geom_hline(yintercept = hT, linetype = "dashed") +
  geom_vline(xintercept = Tc, linetype = "dotted") +
  labs(title = "Rabattement de la nappe (Glover-Dumm)",
       x = "Temps (j)", y = "h (m sous la surface)") +
  theme_minimal()

On attend \(L_{opt} \approx 19\) m. Avec \(L=24\) m, la nappe n'atteindrait 0,30 m qu'après ~3,5 jours — hors norme CPVQ.

Exercice 4 — Abaque \(L(K, q)\) 10 min

Produisez un heatmap de \(L\) (Hooghoudt) pour :

  • \(K\) de 0,1 à 2,0 m/j (50 valeurs)
  • \(q\) de 5 à 25 mm/j (50 valeurs)
  • \(D_0 = 2{,}5\) m, \(H = 0{,}4\) m, \(r_e = 0{,}08\) m
Solution 
solve_hooghoudt <- function(K, q, D0 = 2.5, H = 0.4, re = 0.08){
  L <- 20
  for (i in 1:50){
    de <- d_e(L, D0, re)
    L_new <- sqrt( (8*K*de*H + 4*K*H^2) / q )
    if (abs(L_new - L) < 1e-3) break
    L <- L_new
  }
  L_new
}

grid <- expand.grid(K = seq(0.1, 2.0, length.out = 50),
                    q = seq(0.005, 0.025, length.out = 50)) |>
  mutate(L = mapply(solve_hooghoudt, K, q))

ggplot(grid, aes(K, q*1000, fill = L)) +
  geom_raster() +
  geom_contour(aes(z = L), color = "white", breaks = c(10,15,20,25,30,40,60)) +
  scale_fill_viridis_c(option = "mako") +
  labs(title = "Abaque L (m) — Hooghoudt",
       x = "K (m/j)", y = "q (mm/j)", fill = "L (m)") +
  theme_minimal()

Lecture : pour un critère \(q=10\) mm/j, \(L\) varie de ~10 m en argile (K=0,1) à ~50 m en sable (K=2). Chaque isoligne est un outil de pré-design rapide.

Exercice 5 — Analyse de sensibilité 8 min

On sait que \(L \propto \sqrt{K}\). Vérifiez numériquement et calculez l'élasticité : \[ \epsilon_{L,K} = \dfrac{\partial \ln L}{\partial \ln K} \]

Solution 
K_seq <- seq(0.1, 2, length.out = 30)
L_seq <- sapply(K_seq, solve_hooghoudt, q = 0.010)

# Régression log-log
fit <- lm(log(L_seq) ~ log(K_seq))
coef(fit)
cat(sprintf("Élasticité estimée = %.3f (théorie ~ 0.5)\n", coef(fit)[2]))

ggplot(data.frame(K = K_seq, L = L_seq), aes(K, L)) +
  geom_point() +
  geom_smooth(method = "lm", formula = y ~ x, se = FALSE,
              linetype = "dashed", color = "#961428") +
  scale_x_log10() + scale_y_log10() +
  labs(title = "Élasticité L/K (log-log)",
       x = "K (m/j)", y = "L (m)") +
  theme_minimal()

Conclusion : une erreur de 50 % sur \(K\) entraîne ~22 % d'erreur sur \(L\). D'où l'importance d'une mesure dense de \(K\) (voir TP 4).
