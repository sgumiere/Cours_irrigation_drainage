TP 4 — Mesure de K (terrain)

GAE-3007 · Drainage · Chapitre 4 : Conductivité hydraulique saturée
Université Laval · Prof. Silvio J. Gumiere · Durée : 1 h · Environnement : R
Objectifs pédagogiques
  • Analyser les données brutes de la méthode du trou à la tarière (auger-hole) et appliquer la formule d'Ernst.
  • Analyser les données brutes de la méthode du trou à la tarière inverse (Porchet) et appliquer la formule d'infiltration.
  • Ajuster une droite (régression) pour extraire \(K\) des pentes.
  • Caractériser statistiquement la variabilité de \(K\) sur un champ.
Prérequis R 
library(tidyverse)
Équations

Ernst (trou à la tarière, nappe présente) :

\[ K = \frac{C}{y_0}\cdot \frac{\Delta y}{\Delta t} \]

avec \(C\) un coefficient géométrique fonction de \(y_0, h_0, r, S\) (profondeur imperméable) — environ 0,74 \(r/h_0\) pour les dispositifs standards.

Porchet inverse (sol non saturé, eau descend) :

\[ K = \frac{r}{2}\cdot\frac{\ln\!\left[(2h_1+r)/(2h_2+r)\right]}{t_2 - t_1} \]

où \(r\) est le rayon du trou, \(h_1\) et \(h_2\) les hauteurs d'eau aux temps \(t_1\) et \(t_2\).

Exercice 1 — Auger-hole : ajustement Ernst 15 min

Données brutes d'un essai (nappe stabilisée à 1,15 m ; trou de \(r = 0{,}04\) m, \(h_0 = 0{,}90\) m, \(y_0 = 0{,}40\) m sous le niveau statique ; imperméable à \(S = 2{,}0\) m sous le fond du trou).

ergo_data <- tibble(
  t_min = c(0, 0.5, 1, 2, 3, 4, 6, 8, 10),
  y_cm  = c(40, 37, 34, 30, 27, 24, 20, 17, 14)
)
  1. Tracez \(y(t)\) et \(\ln(y/y_0)\) en fonction de \(t\).
  2. Par régression linéaire, estimez \(\dfrac{1}{y_0}\dfrac{\Delta y}{\Delta t}\) sur les premières 4 minutes (méthode CPVQ).
  3. Calculez \(K\) avec \(C = 0{,}74\,r/h_0\).
Solution 
r  <- 0.04
h0 <- 0.90
y0 <- 0.40
S  <- 2.0

ergo_data <- tibble(
  t_min = c(0, 0.5, 1, 2, 3, 4, 6, 8, 10),
  y_cm  = c(40, 37, 34, 30, 27, 24, 20, 17, 14)
) |> mutate(y_m = y_cm/100)

fit <- lm(y_m ~ t_min, data = ergo_data |> filter(t_min <= 4))
dy_dt_min <- coef(fit)[2]                    # m/min
dy_dt <- dy_dt_min * 1440                    # m/j
cat(sprintf("dy/dt = %.4f m/j\n", dy_dt))

# Coefficient géométrique approché
C <- 0.74 * r / h0
K <- C / y0 * abs(dy_dt)
cat(sprintf("K = %.3f m/j\n", K))

library(ggplot2)
ggplot(ergo_data, aes(t_min, y_m)) +
  geom_point(size=3, color="#003e71") +
  geom_smooth(method="lm", se=FALSE, formula = y~x, color="#961428") +
  labs(title = "Méthode auger-hole — régression Ernst",
       x = "t (min)", y = "y (m)") + theme_minimal()

On obtient typiquement \(K \approx 0{,}4\)–0,6 m/j pour ces données (loam). Attention : utiliser la régression uniquement sur la partie linéaire (4 premières minutes).

Exercice 2 — Porchet : essai inverse 15 min

Sol non saturé, nappe absente. On remplit un trou (\(r = 0{,}04\) m) d'eau jusqu'à 0,60 m, et on mesure la baisse :

porchet <- tibble(
  t_min = c(0, 5, 10, 15, 20, 30, 45, 60),
  h_cm  = c(60, 54, 48, 42, 38, 30, 22, 16)
)
  1. Pour chaque paire consécutive \((t_i, h_i)\), calculez \(K_i\) via Porchet.
  2. Moyennez \(K\) et tracez \(\ln(2h+r)\) vs \(t\) (doit être linéaire).
Solution 
r <- 0.04
porchet <- tibble(
  t_min = c(0, 5, 10, 15, 20, 30, 45, 60),
  h_cm  = c(60, 54, 48, 42, 38, 30, 22, 16)
) |> mutate(h_m = h_cm/100)

porchet <- porchet |>
  mutate(K_int = (r/2) *
           (log((2*lag(h_m)+r)/(2*h_m+r))) /
           ((t_min - lag(t_min))/1440))   # converti min → j
cat(sprintf("K moyen (Porchet) = %.3f m/j\n", mean(porchet$K_int, na.rm=TRUE)))

# Régression sur ln(2h+r) vs t
fit <- lm(log(2*h_m+r) ~ t_min, data = porchet)
pente <- -coef(fit)[2] * 1440                  # 1/j
K_reg <- r/2 * pente
cat(sprintf("K (régression) = %.3f m/j\n", K_reg))

ggplot(porchet, aes(t_min, log(2*h_m+r))) +
  geom_point(size=3, color="#006642") +
  geom_smooth(method="lm", se=FALSE, formula = y~x, color="#961428") +
  labs(title = "Porchet : ln(2h+r) vs temps",
       x = "t (min)", y = "ln(2h+r)") + theme_minimal()

Les valeurs individuelles varient légèrement autour de la moyenne ; la régression sur la série complète est plus robuste.

Exercice 3 — Comparaison : Ernst vs Porchet sur le même champ 10 min

Dans la réalité, les deux méthodes donnent des valeurs voisines mais pas identiques : Ernst intègre \(K_h\), Porchet intègre \(K_v\).

  1. Comparez vos deux estimations.
  2. Estimez un ratio \(K_h/K_v\).
  3. Que recommander pour le design : moyenne géométrique, arithmétique, ou rester avec Ernst seul ?
Solution 
K_Ernst  <- 0.48
K_Porchet <- 0.23
ratio <- K_Ernst / K_Porchet
cat(sprintf("Kh/Kv ~ %.2f\n", ratio))

K_geom <- sqrt(K_Ernst * K_Porchet)
cat(sprintf("K géométrique = %.3f m/j\n", K_geom))

Analyse :

  • Ratio Kh/Kv ~2 typique des loams structurés.
  • Pour l'espacement (Hooghoudt), on privilégie \(K_h\) (Ernst).
  • Pour la recharge / percolation, on privilégie \(K_v\) (Porchet).
  • La moyenne géométrique est une approximation acceptable pour un modèle isotrope simplifié.

Exercice 4 — Statistiques spatiales de K 12 min

Vous avez 25 essais à la tarière sur un plan de 30 ha (données simulées log-normales).

set.seed(1)
K_field <- exp(rnorm(25, mean = log(0.6), sd = 0.7))
  1. Testez la log-normalité (QQ-plot).
  2. Calculez moyenne arithmétique, géométrique, médiane et les percentiles 10/50/90.
  3. Quelle valeur retenir pour dimensionner le plan ? Justifiez.
Solution 
set.seed(1)
K_field <- exp(rnorm(25, mean = log(0.6), sd = 0.7))

stats <- tibble(
  arith = mean(K_field),
  geom  = exp(mean(log(K_field))),
  median= median(K_field),
  p10   = quantile(K_field, 0.10),
  p90   = quantile(K_field, 0.90)
)
print(stats)

qqnorm(log(K_field), main = "QQ-plot ln(K)")
qqline(log(K_field), col = "#961428")

ggplot(data.frame(K = K_field), aes(K)) +
  geom_histogram(bins = 10, fill = "#00549b", color = "white") +
  scale_x_log10() +
  labs(title = "Distribution log-normale de K",
       x = "K (m/j, log)", y = "Fréquence") + theme_minimal()

Recommandation : moyenne géométrique ou médiane comme valeur représentative pour le bloc; utiliser le 10e percentile pour un design prudent (drainage garanti dans 90 % des cellules du plan).

Exercice 5 — Propagation d'incertitude sur L 8 min

À partir de la distribution de \(K\) de l'exercice 4, propagez l'incertitude sur l'espacement L de Hooghoudt (Monte-Carlo, \(q=10\) mm/j, \(D_0=2\) m, \(H=0{,}4\) m, \(r_e=0{,}08\) m).

Solution 
d_e <- function(L, D0, re){ D0 / (1 + (8*D0)/(pi*L) * log(D0/re)) }

solve_L <- function(K, q, D0, H, re){
  L <- 20
  for (i in 1:30){
    de <- d_e(L, D0, re)
    L_new <- sqrt( (8*K*de*H + 4*K*H^2) / q )
    if (abs(L_new-L) < 1e-3) break
    L <- L_new
  }
  L_new
}

L_samples <- sapply(K_field, solve_L, q=0.010, D0=2, H=0.4, re=0.08)

cat(sprintf("L moyen = %.1f m (sd = %.1f)\n", mean(L_samples), sd(L_samples)))
cat(sprintf("P10 — P90 : %.1f — %.1f m\n",
            quantile(L_samples,0.10), quantile(L_samples,0.90)))

ggplot(data.frame(L = L_samples), aes(L)) +
  geom_histogram(bins = 10, fill = "#006642", color = "white") +
  labs(title = "Distribution de L (Monte-Carlo)",
       x = "L (m)", y = "Fréquence") + theme_minimal()

On obtient une plage type 15–40 m. Choisir \(L \approx P_{10}\) pour prendre en compte la variabilité spatiale.
